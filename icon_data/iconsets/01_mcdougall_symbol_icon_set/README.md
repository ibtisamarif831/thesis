# McDougall Symbol/Icon Set

Status: public source material downloaded on 2026-06-09.

Local files:

- `mcdougall_curry_debruijn_1999.pdf`: BURO eprint PDF of McDougall, Curry, and de Bruijn (1999).
- `appendix_pages_png/`: 25 rendered PNG pages from the article appendix, pages 9-33 of the PDF, containing the printed symbols and ratings.
- `extracted_icons_png/`: 239 cropped PNG symbols extracted from the appendix pages. Files are named by appendix item number, e.g. `mcdougall_001.png`.

Source:

- https://eprints.bournemouth.ac.uk/10165/
- https://eprints.bournemouth.ac.uk/10165/1/Measuring_symbol_and_icon_characteristics_mcdougall__curry_%26_debruijn_1999.pdf

Note: standalone original bitmap/vector stimulus files were not found publicly. The extracted PNGs are derived from the public PDF appendix and should be treated as research-use crops, not original source artwork. Request original stimulus files from the author/institution if separate source-quality image files are required.
