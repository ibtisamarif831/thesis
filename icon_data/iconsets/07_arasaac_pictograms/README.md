# ARASAAC Pictograms

Downloaded: 2026-06-14

Source:

- https://arasaac.org
- API metadata: `https://api.arasaac.org/api/pictograms/all/en`
- Static PNG pattern: `https://static.arasaac.org/pictograms/{id}/{id}_300.png`

Local status:

- `metadata/arasaac_all_en.json`: English API metadata.
- `metadata/arasaac_ids.txt`: unique pictogram IDs.
- `png_300/`: downloaded 300px PNG pictograms.
- `metadata/download_failures.json`: unavailable image IDs.

Notes:

- ARASAAC is an AAC pictogram set used in accessibility and communication research.
- Check ARASAAC license terms before redistribution or publication reuse.
