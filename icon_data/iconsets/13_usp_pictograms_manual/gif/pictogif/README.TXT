USP Pictograms
-----------------------------------
The Pictograms Library includes several image files one INDEX.PDF file and this README.TXT file.

The INDEX.PDF file is an index of the pictograms. You must have Adobe Acrobat Reader to view the PDF file. To get Adobe Acrobat reader, download it at http://www.adobe.com/prodindex/acrobat/readstep.html and install.
