# Weekly Task: Icon Perception and Categorisation Direction

Date: 2026-06-14

## Summary

This week focuses on refining the thesis direction around how people perceive and distinguish icons. The key idea is to move from broad questions about icons, images, metaphors, and diagrams toward a more testable study of icon perception, similarity, and categorisation.

The working direction is to investigate whether icon perception can be standardised or measured using visual and structural features. The practical task is to compare different icon sets, identify similarities between icons, and explore whether icons can be grouped into meaningful categories based on visual features.

## Core Concepts

Icons, images, metaphors, and diagrams all represent information, but they do so in different ways:

- Images often share sensory or visual qualities with what they represent.
- Diagrams emphasise structure, relations, and organisation.
- Metaphors make representation possible through parallelism between one concept and another.
- Icons can combine visual similarity, symbolic convention, and structural cues.

For the thesis, the most useful focus is icons because they can be collected, compared, categorised, and tested systematically.

## Research Direction

The current direction can be framed as:

> Can icon perception be made more systematic by identifying visual features that affect similarity, distinguishability, and categorisation?

This can be broken into smaller questions:

- Which visual features make icons look similar or different?
- Are certain shapes, structures, or visual factors consistently distinguishable?
- Can icons be grouped into categories based on similarity?
- Can clustering or hierarchical clustering reveal meaningful icon categories?
- Can analysing multiple icon sets improve how icon perception is standardised?

## Proposed Work For This Week

1. Select several different icon sets for comparison.
2. Start with around 10 different icon sets if enough suitable sources are available.
3. Define visual features that may affect perception, such as:
   - shape;
   - contour;
   - complexity;
   - symmetry;
   - filled vs outline style;
   - number of visual elements;
   - semantic category.
4. Create a categorical system for describing icons.
5. Compare icons by similarity.
6. Test whether icons can be grouped using clustering or hierarchical clustering.
7. Do a small visual examination of the results to check whether the groupings make sense.

## Expected Output

By the end of the week, aim to produce:

- a shortlist of icon sets to use;
- a first list of visual features for comparison;
- an initial category system for icon perception;
- a small sample analysis of icon similarity;
- notes on whether clustering appears useful for the thesis.

## Immediate Next Step

Use the downloaded and planned icon datasets as the starting point. Begin with a small sample rather than the full dataset, then test whether the proposed visual features are enough to distinguish and group icons meaningfully.
